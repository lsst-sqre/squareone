Name the standard in each `/api-aspect` IVOA doc link's accessible label.

The book-icon link beside a curated service now carries a spec-specific
accessible label and tooltip — for example "IVOA TAP docs", "IVOA SIA docs", or
"IVOA DataLink docs" — instead of a generic "IVOA doc". The short standard name
is curated alongside the IVOA URL in the presentation map and threaded through to
the listing, so screen-reader users navigating by links can tell the standards
apart.
