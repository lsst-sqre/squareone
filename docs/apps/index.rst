######################################
Applications in the Squareone monorepo
######################################

Squareone is a monorepo that contains multiple applications.
Each application is developed in separate folder in the :file:`apps` directory of https://github.com/lsst-sqre/squareone and released as a separate Docker image and is intended to be deployed through Phalanx_, Rubin Observatory's Kubernetes deployment platform for the Rubin Science Platform and other Kubernetes clusters.
